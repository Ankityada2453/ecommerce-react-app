# E-Commerce Product Listing Application (React)

## 📌 Project Overview

This project is a responsive **E-Commerce Product Listing Application** built using **React**.
The application allows users to browse products, search for items, filter products by category, and view detailed product information.

Product data is fetched from the **DummyJSON API** and displayed in a modern, user-friendly interface.

---

## 🚀 Features

* Product listing page with responsive grid layout
* Product card showing:

  * Product image
  * Product title
  * Price
  * Discount percentage
  * Rating
* Product details page with:

  * Large product image
  * Product description
  * Brand
  * Category
  * Price
  * Discount percentage
  * Rating
  * Additional images
* Search functionality to search products by name
* Category filtering (smartphones, laptops, fragrances, skincare, groceries)
* Sorting options:

  * Price: Low to High
  * Price: High to Low
  * Rating
* Loading indicator while fetching data
* Fully responsive design for mobile, tablet, and desktop
